# Object Detection with YOLOv8

A real-time object-detection project built on the YOLOv8 deep-learning model.

## Contents
- `Python-Yolo8-object-Detection/` — the detection pipeline

## Use cases
Detecting and counting people and vehicles in real time (people, cars, bicycles, buses, trucks,
motorcycles) — the basis of edge-deployed crowd-monitoring and traffic-analysis systems.

Tech: Python · YOLOv8 · OpenCV.
